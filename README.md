# 🏛️ Portal dos Deputados

Aplicativo desenvolvido em **React Native (Expo)** que consome a API da Câmara dos Deputados para promover transparência e engajamento político.

## ✨ Funcionalidades
- **Listagem Dinâmica:** Perfil completo dos deputados.
- **Transparência:** Detalhes de despesas e órgãos.
- **Notícias:** Feed em tempo real sobre política nacional.
- **Gamificação:** Jogo da memória interativo com os parlamentares.

## 🚀 Tecnologias
- React Native / Expo
- React Native Paper (UI)
- React Navigation (Bottom Tabs & Stack)
- Axios (Consumo de API)
